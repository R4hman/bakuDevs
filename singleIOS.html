Вывести в цикле

Выведите столбец чисел от 16 до 23 с пропуском числа 21. Используйте цикл for, а также сделайте тоже самое с циклом while.

Код для цикла for:

for ($i = 16; $i <= 23; $i++) {
if ($i == 21)
continue;
echo $i;
}

Код для цикла for:

$i = 16;
while ($i <= 23) {
if ($i == 21) {
$i++;
continue;
}

echo $i;
$i++;
}
